Class Node

A Node class that implements NodeType interface

Param

The id of the Node

Param

The X-axis position of the Node (left top corner of the Node)

Param

The Y-axis position of the Node (left top corner of the Node)

Param

The width of the Node

Param

The height of the Node

Param

The background color of the node

Param

A data object that user can specify; possible keys are 'label' and 'custom';

Param

The canvasId of the Svelvet component that the instantiated Node will be on.

Param

The border color of the Node

Param

A boolean set to true if the Node needs to display an image

Param

The src link for the image; image and src are closely tied and a src link is only needed when image sets to true

Param

The color of the text in the Node

Param

The border radius of the Node

Param

An array of node ids that will be grouped as child nodes of this Node. This is for the GroupNodes feature. The current implementation of this feature works one way but not the other (when you drag the parent node, the child nodes will move as a group but when you drag the child node, the parent node would not move along)

Param

The custom class name if user specifies. This is for the custom className feature for Node.
